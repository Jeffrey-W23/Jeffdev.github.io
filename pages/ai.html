INTRODUCTION


This was a project that I completed in my first-year studies at AIE to show my understanding and knowledge of Artificial Intelligence. The project is written in C++ and includes many code examples other than AI, for example, LinkedLists and Associative Arrays.


PROJECT GOALS


The main goal of this assignment was to implement multiple artificial intelligence techniques within a graphical real-time application that runs a simulation with multiple A.I. controlled entities, demonstrating decision making, steering behaviours and pathfinding techniques.

I also tried to make this a compilation piece of past assignments and work I had aceieved thus far at AIE, I was going to require a lot of other scripts (container classes, for example) I had worked on in the previous assignments anyway so I thought a combination like this might be good. I would also like to show all of these other elements visually, so this is something not completed but still planned in the future of this piece.


PERSONAL ACCOMPLISHMENTS


  • Basic Pathfinding
  • Flocking
  • Collision Avoidance
  • Basic Decision Tree
  • Basic Behaviour Tree

LESSONS LEARNED


I didn' know too much about Artificial Intelligence as a topic before I started this assignment, other than it was software techniques to simulate intelligence. So as a result, for example, I learnt a lot about things I didn't even think were AI related at all like Navmeshes and flocking.

During the development of this assignment, I also learnt a lot about different steering behaviours and pathfinding algorithms, I never realised how interesting it was and so in my spare time, I would love to be able to start messing around with that kind of stuff again. It was still a very challenging assignment for me personally but for some reason in my head I expected AI to be way more complicated than it is was, and so I found myself really enjoying this subject.
