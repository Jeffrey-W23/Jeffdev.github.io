Created from boredom and a need for inspiration in games development, The JamJar is a Discord bot designed to help Game developers and Game Jam enthusiasts easily generate game ideas or themes for inspiration or motivation in their development. No need to think or use external sites, generate a random game idea right in discord with your team. Once installed use ?info to get started!

README - INSTALLATION AND CREDITS

Installation and Usage Guide:

  1. Click the Install button at the top of the page. This install button will take you to the Discord Bot Authorization Webpage (discord account and login required).
  2. Read over and confirm the bot permissions required as well as selecting a server under the drop-down menu. Click authorize button, allowing the bot to be added to the selected server.
  3. Navigate to the selected server and type either "?info" or "?help" into a discord text chat to get started with the bot. Enjoy!
