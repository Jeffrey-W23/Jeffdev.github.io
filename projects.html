Introduction:

Zombie Game was one of the first real games that I ever created, I laugh when I see the code today. Regardless I am still very proud of what I achieved back in 2015 with this game, and I will be forever grateful for where it has pushed me.

Zombie Game is a retro-style top-down zombie survival shooter. It's the story of a nameless survivor in an empty, desolate world with nothing but the clothes on his back and a loaded pistol to make it through the day. Learning from my mistakes with previous attempted games (Space Game, I'm very good with names as you can see), I decided to impose a time limit onto myself for when to have the basics of the game done. I also had more free time on my hands compared to the previous project (with school being over and low hours of work).

Project Goals:

The goal for Zombie Game was to have an actual functioning, fun game by the time limit I had given myself. It was to be a top-down shooter with endless spawning zombies and powerups, with the main goal being to survive as long as possible.

Personal Accomplishments:

  • Spawning
  • Ammo system
  • Multiple Game Modes
  • Simple AI
  • Improvements in my pixel art skills
  • Working collision boxes
  • Gained a better understanding of User Interface design
  • Simple scoring system
  • I had made my first real and fun game

Lessons Learned:

I learned what it was like to work under a time limit, I felt the pressure and it made me realise what needed prioritising before others. I learnt an understanding of the limitation of a game engine or platform. The main lesson I would have learnt while creating this game is that when you love what you’re creating, you work a lot faster, have a lot of fun, learn and push yourself and the product a lot further.
